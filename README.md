## Technologies Used

-React.js <br />
-Express.js <br />
-Material UI <br />
-Jest <br />
-Enzyme <br />

## To run the project

to start express

type 'node server.js',  server will be on port 3001, http://localhost:3001

to start react

type 'npm start' and open http://localhost:3000 to view it in the browser.
